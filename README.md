# Riya Sethi Portfolio

Personal portfolio website built with React and Vite.

This portfolio highlights my backend engineering experience across integrations, AI knowledge systems, enterprise tooling, and production-focused backend work.

## Live Site

[https://riya-sethi.vercel.app/](https://riya-sethi.vercel.app/)

## Tech Stack

- React
- Vite
- CSS
- JavaScript

## Sections

- Hero
- About
- Strengths
- Experience
- Engineering Work
- Toolkit
- Engineering Principles
- Contact

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Theme toggle supports dark and light mode
- Built as a frontend-only portfolio
- Deployed using Vercel
